# Строки

В этом проекте были реализованы функции, связанные со строками.
Например: 
- подсчет количества гласных букв в строке;
- приведение всех букв к верхнему и нижнему регистру;
- удаление всех знаков препинания и служебных символов (кроме пробела) из строки;
- проверка, является ли строка палиндромом
